# Example site (consuming @kwcs-mn/eleventy-excellent-core)

## Install
```bash
npm install
```

## Dev
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Site-only customization
Put anything site-specific in:
- `src/site/_includes`
- `src/site/_layouts`
- `src/site/assets/scripts/widgets/...`
